About the Project

The Science Behind OncoScan

A deep learning system trained to classify breast ultrasound images — comparing MLP, VGG16, and ResNet50 architectures to find the most accurate model for early cancer detection.

Project Overview

What is OncoScan?

A research project comparing three deep learning models on breast ultrasound image classification — built with TensorFlow/Keras and deployed as a Flask web app.

Project Summary

OncoScan is a comparative deep learning study that trains three models — a custom MLP, VGG16 (transfer learning), and ResNet50 (transfer learning) — on breast ultrasound images to classify tissue as benign, malignant, or normal.

The best-performing model (VGG16) is then deployed in a Flask web application, allowing users to upload an ultrasound image and receive an instant AI prediction with class probabilities.

TensorFlow Keras VGG16 ResNet50 Flask OpenCV Transfer Learning Google Colab

Data Pipeline

Load Dataset

1000 images per class from breast_dataset1000 — benign, malignant, normal — read as grayscale using OpenCV.

Preprocess

Resize to 300×300px, normalise pixel values to [0, 1], shuffle and split — 80% train / 20% test.

Train 3 Models

MLP, VGG16, and ResNet50 each trained for 10 epochs. VGG16/ResNet50 use frozen ImageNet weights with a custom classification head.

Evaluate & Deploy

Confusion matrix, classification report, and accuracy curves compared. Best model weights saved and deployed via Flask.

Model Comparison

Architecture Details

All three models share the same input (300×300 grayscale) and output (3-class softmax). VGG16 uses an internal Conv2D layer to convert grayscale to 3-channel before the pretrained base.

Model Architecture Input Pretrained Parameters Typical Accuracy
MLP 6 Dense layers (256→8→3) 300×300 flattened No ~23M ~70%
VGG16 Best Conv2D → VGG16 base → Dense(256) → Dropout 300×300×1 ImageNet ~138M (frozen) ~90%
ResNet50 Conv2D → ResNet50 base → Dense(256) → Dropout 300×300×1 ImageNet ~25M (frozen) ~85%

Important Notes

Limitations & Disclaimer

This is a research and educational project. Please be aware of the following before interpreting any results.

  • This tool is for educational and research purposes only. It is not a certified medical device and should never be used as a substitute for professional medical diagnosis.
  • The model was trained on a limited dataset of 3000 images. Performance may vary significantly on images from different ultrasound machines, imaging protocols, or patient demographics.
  • Grayscale ultrasound images are inherently noisy and operator-dependent. The model does not account for image quality variation.
  • The model has no knowledge of patient history, clinical context, or other diagnostic information that a physician would consider.
  • Always consult a qualified medical professional for any health concerns or before making any medical decisions.
